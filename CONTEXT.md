# 唐诗地图（长安诗境）Tang Poetry Map

以交互式地图呈现唐代诗人在长安（今西安）创作或描绘的诗歌与地点。本文件是项目的 ubiquitous language 术语表——仅收录**本项目语境特有**的概念，不含通用编程或地图技术术语（如 GeoJSON、GCJ-02、事件总线等，见 Design Doc）。

## 范围

**长安 Chang'an**:
本项目地图的**核心范围**：唐长安城内坊市宫苑及其京畿近郊（灞桥、渭城、曲江池、乐游原、大雁塔、终南山、华清宫、辋川、樊川等）。
_Avoid_: 西安（今称，仅用于"今称"对照，不指代本项目范围）。

**扩展范围 Extended**:
核心范围之外、与长安主题强相关的远地（阳关、安西、巴蜀、马嵬等）。默认在地图上隐藏，由「扩展范围」开关显示。
_Avoid_: 外地、边塞（含义过宽、不精确）。

## 时期

**时期 Period**:
唐代诗人的四段展示分组：**初唐 / 盛唐 / 中唐 / 晚唐**，统一用于筛选、色板与时间轴分界。年份边界：初 618–712 / 盛 712–762 / 中 762–835 / 晚 835–907。
_Avoid_: 三唐、中晚唐（含糊，无法对应四段）。

## 地点关系

> 注：以下三者描述**诗与地的语义关系**（`placeRelation`），与地点本身的地理归属（核心/扩展，`place.scope`）正交——例如马嵬在地理上属扩展范围，但在《长恨歌》中是"描绘地点"。

**创作地点 Composed place**:
诗篇实际写作所在的地点。
_Avoid_: 出处、来源地。

**描绘地点 Described place**:
诗篇所写的内容地点，可能与创作地点不同（如作于长安而写远方）。
_Avoid_: 题材地。

**关联地点 Related place**:
与诗篇主题相关、但既非创作地也非直接描绘对象的地点，多属扩展范围。
_Avoid_: 相关地点（口语化变体，统一用"关联地点"）。

**约略位置 Approximate location**:
坐标无法精确考据、仅能估算的地点，在地图上以虚线圆样式区别于精确位置。
_Avoid_: 大概位置、模糊位置。

## 诗词

**名句 Highlight**:
一首诗最具标志性的句子（一首可有多句），在标记气泡、列表与详情中突出展示。
_Avoid_: 名句摘录（"名句"即足）、金句。

## 地图

**标记 Marker**:
地图上代表一个地点的可点击图标；**一个地点一枚标记**，若该地关联多首诗则以角标显示诗数。
_Avoid_: 图钉、点位。

**聚合 Cluster**:
地图缩小时将邻近标记合并显示的形态，以减少重叠。
_Avoid_: 簇、分组（"分组"易与筛选分组混淆）。

## 历史地理

**坊 Ward**:
唐长安城网格化布局中的基本居住区划单元，全城约 108 坊。
_Avoid_: 街区。
